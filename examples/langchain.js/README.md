# langchain.js Sigloop Example

Tool-calling agent powered by LangChain.js + OpenAI that can query both:

- `@sigloop/card`
- `@sigloop/api`

## Run

```bash
cd examples/langchain.js
npm install
npm start -- "Summarize card spend + account/contact counts"
```

You can also ask:

- `Get my card limits and latest transactions`
- `List API accounts and contacts`
